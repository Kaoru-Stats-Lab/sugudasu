数字とハイフンだけで打てます（テンキー／スマホの数字キーボード想定）。900-1800 のように貼るだけで合計（h:mm · 10進小数）と時給換算が出ます。解釈が違ったら行の ↺。

残業代や法定労働時間の判定は行いません。単純な時間合計と時給換算のみです。

入力した時間・時給はブラウザ内でのみ処理され、外部には送信されません。

入力

正規の打ち方（コロン不要 · 数字と - だけ）

  • 出勤〜退勤 … 900-1800 / 0900-1800（迷ったら4桁）
  • 夜勤 … 22-6 / 2200-0600
  • 実働だけ … 130（1時間30分）· 90（90分）· 8（8時間）· 1.5（時間の小数）

1行=1日分。行末メモ（900-1800 遅番）は無視。休憩は引いたあとの時間を書いてください。Excelのコロン付きも読めますが、手入力では打たなくて大丈夫です。

注意: 9-1 は「9→13時」と「9→翌1時」がぶつかります。迷ったら 0900-1300 か、結果の ↺ で切替。

残業代・割増賃金・休憩控除・複数人集計は対象外です。

時給計算のよくある質問

勤務時間の単純合計と時給換算に特化したツールです。給与計算（社会保険・税金・残業代）ではありません。

コロンは必要ですか？どう書けばよいですか？

不要です。 正規は数字とハイフンだけです。出勤〜退勤は 900-1800 / 0900-1800、夜勤は 2200-0600、実働だけは 130（1時間30分）・ 90（90分）・ 8（8時間）。結果の表示だけ 09:00 形式になります。Excelからコロン付きを貼っても読めます。短い 9-1 はぶつかりやすいので、違ったら ↺ で切替えてください。

入力した時間や時給はサーバーに送信されますか？

送信されません。計算はブラウザ内だけで行われ、登録やログインも不要です。

日またぎ勤務（例: 22-6）は計算できますか？

できます。22-6 や 2200-0600 のように、終了が開始より早い場合は翌日跨ぎとして扱い、警告付きで合計に載せます。違ったら ↺ で同日解釈に切り替えられます。

休憩時間や残業代の計算はできますか？

このツールでは対応していません。休憩は事前に差し引いた時間を入力し、残業代・割増賃金は勤務先の計算ルールで確認してください。

エラー行があると全体の計算は止まりますか？

止まりません。エラー行は一覧で表示しつつ、その行だけ合計から除外して他の行は計算を継続します。

支給額はどのように丸めていますか？

仕様どおり、円未満は切り捨てです。表示はあくまで概算なので、実際の給与額は勤務先の計算方法をご確認ください。

給与計算ツールですか？

いいえ。勤務時間の合計と時給換算のみです。社会保険・税金・手取り・残業代の自動計算は行いません。
