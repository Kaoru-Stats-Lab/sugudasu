変換サイトに貼れない名簿を整える — 都内オフィスの総務OL編

丸の内近郊のオフィスビルで働く総務・営業事務の方を想定した読み物です。顧客名簿や社員リストを Excel からコピーしたあと、「無料の全角半角サイトに貼るのは社内規程でアウト」と言われて手が止まった経験は、珍しくありません。

本記事では架空の事例を通して、ブラウザ内だけで列を整える場面と、全角半角整えの使いどころを整理します。

この記事でわかること

  • どんな職種の OL が、どんな場面で名簿整形に困るか
  • Excel 関数 · 外部コンバータ · ブラウザ内処理の使い分け
  • 行ズレ・先頭ゼロ・全角メールなど、事故りやすいポイント
  • 規程説明のときに「データの行き先」をどう伝えるか

美咲さんは「エンジニアではない」けれど、現場の最後の1マイル（リストを渡す直前の整形）を任されがちです。開発部門にチケットを切るほどでもないが、手作業は毎週つらい — そんな層が 全角半角整え の主な読者です。

1日のなかで助けられる場面（4つ）

場面A — 9:10 · ECサイトへの一括会員登録前

営業部から Slack で届いた Excel。「この顧客200件、今週中にモール側に登録お願い」。E列のメールアドレスをコピーしたら、ＴＡＮＡＫＡ＠ＥＸＡＭＰＬＥ．ＣＯ．ＪＰ のように全角英数だらけ。モールの管理画面は半角しか受け付けない。

美咲さんが試した退路:

  1. Excel で補助列に ASC を入れる案 → ブックが共有中で、触ると他シートの参照が壊れそうで躊躇。
  2. 検索で出てきた無料コンバータ → 規程で「顧客メールを外部に送るな」とリマインドされ、タブを閉じる。
  3. 全角半角整え を開き、列を貼付 → EC登録プリセット → 行数バッジが「200 → 200」のまま → コピーして管理画面へ。

※本ツールは Excel の代替ではありません。毎週同じブックで補助列を回すなら、テンプレ化の方が向くことが多いです。

場面B — 11:00 · 新人研修の班分け前

人事から「来週の研修、32名の班分けお願い」。名簿は問題ないが、氏名の全角スペースがバラバラ（「田中　美咲」「山田 太郎」混在）。班分け に渡す前に、姓名の空白を揃えたい。

全角半角整え の 姓名スペース プリセットで1列整形 → 行数確認 → 班分け へ。班分け結果を説明するときは、シード番号をメモしておくと「なぜこの組み合わせか」を後から答えやすくなります。

場面C — 14:30 · メール配信リストを1行にまとめる

マーケ支援で「BCC用にメールアドレスをカンマ区切り1行で」。縦に並んだ300件を、手でカンマを入れるのは絶望的。

改行→カンマプリセットを使うと、意図的に行数が1行に変わる。このときだけ「入力 N 行 → 出力 1 行」とバッジが変わるのは正常です。コピー前に必ず目視確認してください。

場面D — 16:00 · 社員番号の先頭ゼロ

協力会社向けの連携ファイルで、社員番号 00123 が Excel に貼った瞬間 123 になる事故が先週あった。美咲さんは今度こそ怖い。

全角半角整え は数値型に変換せず文字列のまま処理します。画面上部に「0始まりコードを維持します」と出る行もあります。それでも貼り先の Excel が「数値」列だとゼロが落ちるので、貼り付け先の書式もセットで確認が必要です。

開発者やデータエンジニアは、Python や Excel マクロで済ませることが多く、本ツールの主戦場ではありません。「チケットを切るほどでもないが、規程は厳しい」事務職が中心です。

3つの手段の使い分け

手段向くとき不向きなとき規程説明
Excel 関数（ASC/TRIM 等）同じブックで完結 · 再現性重視共有ブックを触れない · 列だけ別画面へデータは社内ファイル内
無料の Web コンバータ個人メモ · 非機微テキスト顧客 · 社員の実データ第三者サーバーへ送信されうる
全角半角整え列コピーの途中 · 500行以内 · 説明しやすさ毎週千行の ETL · 複数列同時結合ブラウザ内完結（SUGUDASU の約束）

情報システム部に聞かれたときは、「当社ドメインへテキストを POST しない」ことを DevTools の Network タブで見せられると、説明が早い場合があります。最終判断は貴社のセキュリティ担当へ。

美咲さん流 · 5分チェックリスト

  1. Excel で1列だけコピー（余計な見出し行を混ぜない）
  2. 全角半角整え に貼付 · 行数が 500 以下か確認
  3. 用途に合うプリセット（EC / 名簿 / カンマ結合 等）を選択
  4. 入力 N 行 → 出力 M 行 を確認（通常は N=M。カンマ結合だけ例外）
  5. コピー → 貼り先で先頭数行と末尾数行を目視

閉域でテスト用の架空データが先に必要なら、テストデータ から CSV を生成し「全角半角整えで開く」導線も使えます。本番名簿と混ぜないよう、タブを分けてください。

よくある質問

都内の中堅商社でも、本当に「変換サイト禁止」はある？

業種・規模は問いません。取引先データや個人情報を扱う部署では、「無料ツールに貼らない」ルールは珍しくありません。禁止の有無は貴社の規程を正としてください。本記事は一般的な事務現場の描写です。

Excel だけで十分なのに、わざわざ Web ツールを使う意味は？

意味があるのは「シートを壊したくない」「列だけ別の画面に持っていく」ときです。毎週同じマクロで回すなら Excel テンプレが正解です。

スクショの顧客名はどうする？

テキスト列は本記事の 全角半角整え、画像のマスキングは マスク（ぼかし · 黒塗り · 背景同色）。スタンプだけでは機密は隠れません。
