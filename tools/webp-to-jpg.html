WebP→JPG/PNG · 画像をアップロードしない

iLoveIMG や Convertio もブラウザで使えますが、画像は運営サーバーへアップロードされます。 本ツールは 端末内だけで WebP→JPG/PNG。社外秘スクショ・顧客素材向け（最大20枚）。

    広告枠

    WebP→JPG/PNG のよくある質問

    iLoveIMG・Convertio・PDF24 と何が違いますか？

    それらは画像を運営サーバーへアップロードしてから変換します（Convertio は変換後にサーバーから削除、PDF24 は約1時間で削除、と説明しています）。SUGUDASU は端末のブラウザ内だけで Canvas 変換し、画像ファイルの POST は発生しません。F12 → ネットワークで確認できます。数百枚の一括 ZIP が必要ならアップロード型の方が向いています。

    WebPをPNGやJPEGに変換できますか？

    はい。一覧から出力形式を選び、変換後に1枚ずつダウンロードできます。ExcelやWordに貼れない WebP も PNG / JPEG にすれば使えることが多いです。

    画像は外部に送信されますか？

    いいえ。すべて端末のブラウザ内でデコード・再エンコードされます。未公開のスクショなど、アップロード型サイトに載せたくない場合向けです。

    Web 上の WebP を URL から変換できますか？

    URL 欄または URL をコピーして Ctrl+V で試せます。ただし多くのサイトはブラウザの CORS 制限で直接取得できません。その場合は「名前を付けて保存」してドロップするか、ページ上で画像をコピーして貼り付けてください。処理は端末内だけで行い、SUGUDASU のサーバーへ送信しません。

    PNGやJPEGをWebPに変換できますか？

    いいえ。本ツールは WebP を PNG/JPEG に戻す用途向けです（Excel 貼付 · 旧システムへの画像登録など）。PNG/JPEG を WebP に圧縮する用途は別の Pain です。

    透過WebPをJPEGにするとどうなりますか？

    JPEGは透過非対応のため、背景は白で塗りつぶされます。透過を残す場合は PNG を選んでください。

    PDFや一括圧縮はできますか？

    できません。PDF 結合・高度な編集は専用サービス向けです。本ツールは WebP 入力のみです。
