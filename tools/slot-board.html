よくある質問

どんな場面で使いますか？

評価会議・昇格候補・受賞候補など、人数の上限が決まっている会議向けです。Excelの評価案を貼り付けると、枠を超えている評価ランクがすぐに分かります。

人事考課の相対評価にも使えますか？

使えます。「S評価は3人まで」のように人数上限がある評価であれば、部署別・等級別など枠を分けて配置状況を確認できます。

Excelはそのまま貼れますか？

名前と評価ランクをコピーするだけです（互換の一括貼付）。枠だけ先に決めるときは Box1、候補だけは Box2 を使います。多少の空白や余分な文字は自動整理します。

データは送信されますか？

送信されません。処理はブラウザ内だけです。同じ端末では IndexedDB に残ります。別PCへ持ち運ぶときは復元コードまたは JSON / TSV をコピーしてください。

自動で配置してくれますか？

しません。人数の集計と上限超過の表示だけです。配置はドラッグ（またはタップ）で人間が決めます。

うまく読み取れない行があったらどうなりますか？

読み取れなかった行は、削除せずに「不明」として残します（一括貼付時）。人数が合わない場合は、貼り付けた元データを確認してください。

まだ人数の上限が決まっていなくても使えますか？

使えます。上限人数の欄を空欄にしておけば、人数の制限なしで配置だけ確認できます。上限が決まったら、あとから数字を入力してください。

「復元コードをコピー」は何のためのものですか？

現在の配置状態を1つの文字列に変換してコピーする機能です。SUGUDASU側にクラウド保存されるわけではなく、メールやチャットの下書きに貼り付けて、あなた自身が持ち運ぶためのものです。同じ文字列を貼り付ければ元の配置に戻せます。日をまたぐ・別PCでは JSON ダウンロードも使えます。
