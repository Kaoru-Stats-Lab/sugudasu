## 2. UI · 入出力

### 入力

- 複数行テキストエリア（コピペ）または行追加 UI
- 時給（円）フィールド

### パース優先順位

1. `h:mm` / `hh:mm`（例 `1:30` `02:45`）
2. 10進数（例 `1.5` → 1時間30分）
3. 日本語（例 `90分` `1時間30分`）

不正行は `0:00` またはスキップ + 薄い警告。

### 出力

- **総労働時間:** `58:30` 形式（**24時間超えも累計** — Excel `[h]:mm` 罠の回避）
- **10進時間:** 例 `58.5時間`
- **概算支給額:** 時給 × 総分（端数 **切捨て** 明示）
