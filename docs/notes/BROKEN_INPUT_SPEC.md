## 3. V1 Must

カテゴリ 4 つ（固定プリセットのみ）:

| カテゴリ | 目的 | MVP プリセット |
|----------|------|----------------|
| 長さ | 文字数制限・CSS崩壊 | 100英 / 100日 / 1000 / 10000（プレビュー省略・タイトル＋確認用のみ） |
| 多言語 | 方向・結合・幅・フォント | 説明は「確認: …」。日英・中簡繁・韓・タイ・アラビア（数字混在）・ヘブライほか |
| 文字 | 文字数ズレ・見えない・分解 | 絵文字 / 家族 / 異体字 / 見えない文字 / 分解「が」 |
| 記号・制御 | 改行・エスケープ・制御 | 改行 / タブ / HTML記号 / NULL文字入り（貼り付け先の反応観察・保証しない） |

**設計必須:** `preview`（画面）と `value`（コピー）を分離。見えない文字は見た目が普通でもコピーにゼロ幅が入る。

コピー成功: toast `✓ コピーしました`（約 0.8s）

SEO 説明に拾う語（タイトルには載せない）: 異常系テスト · 境界値テスト · 入力値チェック · 文字化け確認 · 最大文字数テスト
