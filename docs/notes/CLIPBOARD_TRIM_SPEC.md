# SUGUDASU 余白トリム — 仕様 SSOT（v0.1）

**更新:** 2026-07-20  
**ステータス:** v0.1 実装  
**id:** `clipboard-trim` · URL `/clipboard-trim`  
**関連:** image-trim（指定サイズ切出し）· mask · watermark  

> **一言:** クリップボードに入った画像の余白を落とし、PNG をクリップボードへ戻す。資料貼付の「中継」— 保存概念なし。非送信。

## 立ち位置（プロダクト）

| 隣 | 役割 |
|----|------|
| **image-trim** | 指定サイズ（枠）に**収める** |
| **余白トリム** | 周囲の不要余白を**限界まで削る** · クリップボード専用 |
| **mask** | 貼付前後の**機密消し** |

**最強ジャーニー:** ウェブ右クリックコピー / スクショ → Ctrl+V → PNGコピー → PowerPoint Ctrl+V。ファイルを経由しない。

## リレー（画面リンク）

| 方向 | ツール | いつ |
|------|--------|------|
| 前 | **mask** | 機密・個人情報が写っている |
| 後 | **image-trim** | 余白のあと、指定サイズ（枠）に合わせたい |
| 後 | **watermark** | 社外配布・転載防止の透かし |
| 前 | **webp-to-jpg** | WebP が貼れないとき（変換してから貼る） |

`data/relations.json` · 各ツール `sg-tool-lead` / `sg-tool-lead--meta` と同期。

## Must

| 項目 | 内容 |
|------|------|
| 入力 | `.sg-file-drop`（Ctrl+V · DnD · クリック）— mask / image-trim と同型 |
| 処理 | Canvas · 透明余白 trim。全面不透明なら四隅色で余白除去 |
| 出力 | 「PNGをコピー」1ボタン · Clipboard API |
| 非送信 | `data-sg-privacy-badge` subject=`画像` |
| UI | DnD · プレビュー · コピーボタン · メッセージのみ |

## OUT（v0.1）

- ダウンロード · 保存
- 手動トリムハンドル · リサイズ · 回転 · フィルター · 色補正
- AI / OCR / 背景除去 · 設定画面 · Undo · 履歴 · ダークモード

## Non-goal

指定サイズ切り出しは `/image-trim`。機密消しは `/mask`。

## エラー文言（固定）

| 条件 | 文言 |
|------|------|
| 未貼付でコピー | `画像を貼り付けてください` |
| Clipboard API 非対応 / 失敗 | `このブラウザでは画像コピーに対応していません` |
