## §5 ギャップ分析

- **海外競合が強い:** WebSocket 完全同期 · 全画面タイマー（緑黄赤）· 秒単位放送管理
- **コアが勝てる隙間:** 登録不要 · 非送信 · 日本語 · スマホ · Excel 代替の Day0 aha!
- **Sync が勝てる隙間:** 手動反映既定 · イベント単位課金 · 会場通信不安への耐性

## §6 MVP 推奨（仕様に反映済み）

### 行メモ `note`

- 最大 **100字** · 一覧は `...` 省略 · タップで全表示
- コピー形式: `【10:15–10:30】アジェンダ名 （備考：〇〇さん移動）`
- 印刷: 備考列として出力

### 現在時刻

- **固定ヘッダー**にデバイス実時間
- リスト内: 現在に最も近い行間に `—— 現在時刻（10:22）——` インジケータ（スクロール追従）

### 残り時間（sticky フッター）

- **デフォルト:** 進行中コマの残り
- **行タップ:** その行の**開始まで**のカウントダウン（例: `お昼休憩まで あと 45:12`）

### MVP でやらない（競合にあるが後回し）

- 秒単位 `hh:mm:ss`（分単位で十分）
- 役割別マルチ列（音響・照明フィルタ）
- 外部モニター専用タイマー出力（Stagetimer 型）

## 提督追記（2026-06-24）— UI 参考・フォーク方針

| 製品 | 位置づけ |
|------|----------|
| **[Rundown Studio](https://rundownstudio.app/)**（例: [サンプル rundown](https://app.rundownstudio.app/rundown/JpyhODPhqUyiRPTUiAaO)） | **UI/UX の主参考** — 開始·所要分·連動終了·早終了表示·Hard Start |
| **[live-board](https://github.com/tsubasagit/live-board)** | **参考にしない（本体）** — 内容が違う。「今何番」同期表示 · 次へボタン型。MIT だがフォークして直すより自前 |

### Rundown 画面から SUGUDASU が取るもの（MVP）

| Rundown の要素 | SUGUDASU 対応 |
|----------------|---------------|
| Start · Duration · Description 列 | `startAt` · `durationMin` · `title` + `note` |
| Hard Start（★） | **アンカー行** `anchored` |
| −1m / +1m ジャンプ | **−5 / +5 分**（スマホフッター） |
| Planned end / End of rundown | フッター **終了予定時刻** |
| Rundown ends **45m early** | **目標終了との差分**（v1.1 · 早終了/遅延） |
| Private notes 列 | `note`（100字） |
| Start show · 本番タイマー | **Phase B / Sync** — MVP は編集モード優先 |

### 意図的に取らない（Rundown にはあるが）

- カスタム列 · リッチテキスト · 画像セル · ドロップダウン列
- ダーク放送 UI（SUGUDASU は `sg-*` ライト）
- 秒単位 UTC クロック · Auto-Scroll 本番モード（最初は後回し）

### Rundown LP・製品画面から追加（提督スクショ 2026-06-24）

[Rundown Studio](https://rundownstudio.app/) マーケ画面が示す **製品の芯**（live-board とは別物）。

| Rundown のパターン | SUGUDASU への落とし方 |
|-------------------|----------------------|
| **表＋タイマー**（スプレッドシート × 放送時計） | 左: 行編集 · 右/タブ: プレビュー＋**大きい残り時間** |
| 列: Start · **Dur** · **End** · Title · **Notes** | `startAt` · `durationMin` · `endAt` · `title` · `note` |
| **赤い「今」ライン**（表横断） | プレビューに **現在時刻インジケータ線** + 進行中行ハイライト（emerald/amber · 赤は避けても可） |
| 現在コマの **巨大カウントダウン** + **Next:** 次タイトル | sticky フッター上部 or プレビュー先頭（§5-1b） |
| 所要分変更 → **後続がリップル** | `recalcTimeline`（コア） |
| **多端末リアルタイム**（常時自動） | **Sync** — **手動反映既定**（差別化） |
| **閲覧専用リンク**（ステージモニター用タイマーのみ） | Sync 閲覧 URL · 編集 UI なし |
| モバイルは **現行＋タイマー＋直近数行** に簡略 | スマホ **プレビュータブ** はフル表より **現行・次2コマ** 優先表示可 |
| PDF/Excel 出力 | 印刷 · TSV コピー |
| 共同編集（Google 表級） | MVP 外 · Sync は 1編集者多閲覧 |
