## 2. 機能要件（提督 · 2026-06-26）

### 2-1. 日ベースと時間ベースの両方

| モード | 用途 | 連動の単位 |
|--------|------|------------|
| **日ベース** | 週次工程 · マイルストーン · 制作納期 | **日数**（開始日 + N 日 → 後続行の開始日が連鎖） |
| **時間ベース** | 当日進行 · 設営 · 分刻み | **分**（既存 `timeline-engine` 相当 · 時刻 HH:mm） |

**方針:**

- 1プロジェクト（シート）で **モードを選ぶ**、または **セクション単位**で切替 — 実装時に dogfood で決める（両方同一シート混在は v2 検討）。
- イベント進行 T13 は **時間ベース専用**のまま。Schedule は **日ベースを第一級**にする。

### 2-2. 列は無限に増やせる

| 項目 | 内容 |
|------|------|
| **固定列** | 行ID · 連動用の開始/終了（日 or 時刻）· 所要（日数 or 分）— エンジンが読む最小セット |
| **ユーザー列** | **ラベル自由 · 数に上限なし**（担当 · 工種 · クライアント · メモ · ステータス …） |
| **データ形** | `columns[]` + `rows[].cells{ colId: value }` — TSV の固定4列モデルとは **別 SSOT** |
| **上限** | 実装は **ソフト警告**（例: 列 30 · 行 500）のみ。ハード cap はコスト防衛用に Sync 側で検討 |

**Excel っぽさの核:** 列を足しても **連動列だけ壊れない**。

### 2-3. 印刷 — **A3 必須**

| 項目 | 内容 |
|------|------|
| **用紙** | **A3**（横推奨 — 列が増える前提） |
| **参照** | 既存は T13 = A4 縦（`TIMELINE_IMPLEMENTATION_SLICES`）— Schedule は **別 CSS** `@page { size: A3 landscape }` |
| **出力** | ブラウザ印刷 → PDF。列幅は印刷プレビューで折り返し/縮小のルールを定義 |
| **画面** | 編集と印刷プレビュー分離（`invoice` / `group-split` 型は維持） |

### 2-4. Excel — **Import なし · Export あり**

| | 方針 |
|--|------|
| **Import** | **やらない**（v1）。コピペ取込も **スコープ外** — 手入力 or アプリ内複製 |
| **Export** | **xlsx ダウンロード必須**（全列 · 連動後の開始/終了を反映） |
| **TSV コピー** | あってよい（SUGUDASU 文化）が **正本は xlsx Export** |
| **理由** | パース・結合セル・マクロ地獄を避ける。出口だけ Excel に合わせる |

### 2-5. やらない（維持）

クリティカルパス · 依存矢印 · リソース平準化 · ERP 代替 · ガント横棒 DnD · **Excel Import**

## 3. イベント進行（T13）との関係

| | **T13 イベント進行** | **Schedule 工程表** |
|--|----------------------|---------------------|
| 時間軸 | **時間**（当日） | **日 + 時間**（両モード） |
| 列 | 固定（開始/終了/分/タイトル/note） | **可変列 ∞** |
| 印刷 | A4 縦 | **A3**（横） |
| Excel | TSV コピー中心 | **xlsx Export のみ** |
| Import | JSON 下書き（コア） | **なし** |
| 課金 | イベントチケット | **¥200/月・アカウント** |
| URL（案） | `sync…/timeline` | `sync…/schedule` |

## 5. 実装順（提督 dogfood 優先）

1. **データモデル** — 可変列 + 日ベース連動エンジン（`schedule-engine.js` 新規が妥当）

3. **UI** — 列追加 · 行編集 · A3 印刷プレビュー
4. **xlsx Export** — クライアント生成（依存は最小1本 · 要 `package.json` 判断）
5. **Sync** — ルーム · ¥200/月 Checkout · 閲覧 URL
6. **LP** — `schedule`（ガントと名乗らない）
