## 骨子（Zenn #11 から移す節）

### 1. リード

- いまどの会社も SNS CP を回す · 幹事は法務なしが普通
- 抽選の公平性（Zenn #11）と **企画の法令適合** は別問題
- fair-draw Phase 0 は **問い合わせ推奨の下準備** — 合法判定ツールではない

### 2. Phase 0 の使い方

- 揉め事カード P01–P15 · 告示ベース数値
- 推奨フロー（フラグ → 相談 → 抽選 → 証跡）
- ChatGPT に聞かない理由

### 3. 公的先例 — 晋遊舎

- [課徴金公表 PDF（2022-08）](https://www.caa.go.jp/notice/assets/representation_220805_01.pdf)
- 抽選不正ではなく **表示と履行の乖離**
- fair-draw は発送まで見ない（限界）

### 4. 消費者庁に聞いていいのか

- [相談窓口](https://www.caa.go.jp/policies/policy/representation/contact) · 事業者 4→1→1
- 気軽ではない · Q&A 先読み · 回答に時間 · 直前NG
- 188 / 消費生活センターは消費者向けが主

### 5. 社内法務がない場合

1. Q&A + Phase 0  
2. 消費者庁（企画前・数週間前）  
3. 高リスクのみスポット弁護士  
4. Wiki に Q&A 蓄積  

### 6. SNS CP ナレッジ

- フォローRT懸賞は？ · X+IG 口数2倍（P13）
- 社内先例が最強の法務代替
- チェック結果 + 相談回答を1ページに
