## 1. ペルソナ · Job

### 1-1. 主ペルソナ

| ID | 姿 |
|----|-----|
| **P-M（マニュアル担当）** | 社内手順書・ヘルプをたまに作る。操作録画は手元にある |
| **P-B（併せて）** | 不具合報告・社内システム説明で画面キャプチャが要る。社外秘画面を変換サイトに上げられない |

### 1-2. Job

| # | 声 | 仕様 |
|---|-----|------|
| J1 | 「一時停止スクショだと再生UIが映り込む」 | 動画キャンバスから純粋なフレームを PNG 化 |
| J2 | 「社外秘の操作画面を外部に送りたくない」 | ローカル Blob のみ · 非送信 |
| J3 | 「動くGIFはマニュアルに向かない」 | PNG のみ |
| J4 | 「ちょうどホバーした瞬間を止めたい」 | コマ送り/戻し（前後フレーム相当）＋キーボード |
| J5 | 「連続で抜くとファイル名が迷子」 | `{安全な元名}_{mm}m{ss}s.png`（ゼロ埋め） |

### 1-3. 成功定義（30秒）

1. MP4/WebM をドロップ  
2. シーク（＋コマ送り / ←→）で止める  
3. 「現在の画面をPNG保存」  
4. トースト確認 → マニュアルに貼る

## 2. 境界（防波堤）

> **動画から説明用の静止画（PNG）を1枚取り出すツールです。GIF・動画編集は対象外です。**

| やること | やらないこと |
|----------|----------------|
| ローカル動画 → 現在フレーム PNG | GIF / WebM / MP4 書き出し |
| シーク · 前後フレーム相当 | トリミング · カット · 字幕 · OCR |
| ブラウザ内 Canvas | クラウド変換 · URL 読込 |
| 透かしなし · 画質向上なし | Tango 型 · AI ベストフレーム · モザイク |

キャプチャ後の黒塗り・注釈は既存 **マスク**、サイズ合わせは **画像切り出し**。

## 3. Must（v0.1）

| 機能 | 内容 |
|------|------|
| 入力 | `<input type="file" accept="video/mp4,video/webm">` または D&D。**Blob URL のみ**（URL 読込禁止） |
| 再生UI | `<video>` プレビュー。PNG にネイティブコントロールが映らないこと（カスタムシーク推奨 · `controls` は出さないか非表示） |
| 微調整 | **戻る / 進む**（おおよそ ±1フレーム相当の時間ジャンプ）＋ **キーボード左右矢印**（フォーカス時）。ボタンに ← / → ヒント |
| 出力 | 現在表示を Canvas → PNG ダウンロード（**元フレーム解像度**） |
| 保存 CTA | 文言 **「現在の画面をPNG保存」** |
| 保存後 | 短いトースト: `{ファイル名} を保存しました` |
| ファイル名 | `{sanitize(元名)}_{mm}m{ss}s.png` · **分・秒は2桁ゼロ埋め**（例 `01m05s`）。禁則文字（`/ \ : * ? " < > |` 等）は除去または `_` |
| 非送信注記 | 「サーバーに送りません」1行 |
| コーデック免責 | 「ブラウザが再生できない形式では表示できません。MP4（H.264）または WebM を推奨します。」 |
| 重い動画 | ハード上限なし。注記:「大きな動画は端末によって処理できない場合があります」（数値の目安は FAQ のみ可） |
| 画質誤解防止 | 「画質を良くするものではありません。元の解像度のまま保存します。」（短く） |

### フレーム精度（受け入れ用注記）

HTML `<video>` に厳密なフレーム API 保証はない。実装は **短い時間ジャンプ（例: 1/30 秒前後）** で近似する。  
仕様文言: **「前後フレーム相当。動画の fps・ブラウザにより誤差が出ることがあります。」**

### 技術受け入れ

- 処理はローカル Blob · Network に動画アップロードなし  
- コードレビューで確認

## 4. Should / OUT

### Should（v0.1 に無くてよい）

| 項目 | 理由 |
|------|------|
| クリップボードへ PNG コピー | ブラウザ差 → まず DL |
| 秒数の数値直入力 | シーク＋コマで足りる（第2: GPT OUT寄り / Grok Must → Should 維持） |
| 複数コマ ZIP | v0.2 以降 |
| マスクへの「このPNGを開く」導線 | あるとよい |

### OUT

GIF · 動画編集 · トリミング · 字幕 · OCR · AI 抽出 · Tango · URL 読み込み · ffmpeg.wasm · MOV/HEVC 完全保証 · ハードな MB 上限での拒否。

## 8. 受け入れ条件（実装時）

1. ローカル MP4/WebM を開き、現在画面を PNG 保存できる  
2. 前後フレーム相当の操作（ボタン＋左右矢印）がある · 精度非保証の注記がある  
3. ファイル名がゼロ埋め時刻付き · 禁則文字が落ちている  
4. 保存後にトーストが出る  
5. DevTools で動画の外部アップロードがない  
6. FAQ: GIFにならない · コーデック · 非送信 · 大きなファイル
