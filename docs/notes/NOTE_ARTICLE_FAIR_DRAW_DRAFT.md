### 本文

## 抽選の公平と、企画の法令適合は別問題

懸賞キャンペーンで幹事が聞かれるのは、だいたい次の2種類です。

1. **「くじは公正だった？」** — 名簿 · 番号 · 再現性 · 説明責任  
2. **「企画は法令的に大丈夫？」** — 景品の上限 · 応募条件 · 表示と履行  

1 だけ整えても、2 でこけると企画全体の信頼は終わります。逆に、景表法をきちんと見ても、抽選が説明できなければ「ヤラセでは？」と疑われます。

抽選の公平性・Excel `RAND()` からの卒業・証跡3点の話は Zenn 側で書いています（公開後ここにリンク）。**この note では 2 のほう** — 特に **法務部がない組織** が、何を読み、誰に聞き、社内に何を残すかに絞ります。

## ChatGPT に「大丈夫？」と聞かない理由

法務がいない現場では、景品表示法を ChatGPT に聞く幹事をよく見ます。私も試しましたが、**断定と根拠の薄い答え** が混ざりやすい。ハルシネーションで「問題なし」と言われても、後から説明できません。

代わりに必要なのは次の2つです。

- **告示や Q&A に根拠がある一次チェック**（数値 · パターン）  
- **最終判断は専門家に委ねる** — ただし、ゼロから聞くのではなく **論点を整理してから** 聞く  

[SUGUDASU 抽選の景品チェック](https://sugudasu.com/fair-draw?tab=check&utm_source=note&utm_medium=social&utm_campaign=note_fairdraw_prizelaw) は、合法判定ツールではありません。**問い合わせることを推奨するための下準備** として作りました。黄/赤フラグが出たら、そのまま公開せず確認へ。結果は「合法」「違法」とは言いません（[免責事項](https://sugudasu.com/disclaimer)）。

## Phase 0 の使い方 — 推奨フロー

抽選の前に、企画が **揉めやすいか** を整理するタブです。

1. 企画条件を入力し、**揉め事カード**（P01–P15 等）と **告示ベースの数値** を確認  
2. 黄/赤フラグが出たら、**専門家に問い合わせ**（ツールは「聞くべき論点」のメモ代わり）  
3. フラグがなくても、初めての懸賞 · 高額景品 · 複数 SNS などは **念のため相談を推奨**  
4. 確認が取れたら抽選へ — 証跡 PDF に企画条件を残す  

**揉め事カード** は、複数コース · 複数 SNS · 購入条件つきなど、公表事例から抽出した **注意パターン** です。**数値** は LLM ではなく告示ベースのルール表でチェックします。

条文解釈と **企画が範囲内かの最終判断は専門家に任せる** — これが製品の割り切りです。

## 晋遊舎の課徴金 — 抽選以前の「表示と履行」

くじの不正ではなく、**表示と実態の乖離** で有名なのが、[消費者庁の公表資料（2022年8月）](https://www.caa.go.jp/notice/assets/representation_220805_01.pdf) にある **株式会社晋遊舎** の事例です。懸賞付きパズル雑誌 **63誌** に対し、景品表示法に基づく **課徴金納付命令**（合計約1,231万円）が出ています。

誌面では締切後、相当期間内に当選者へ賞品が届くかのように見えたのに、実際の発送は **応募締切から最長で約4年近く遅れた**。2021年の措置命令で優良誤認・有利誤認が認められたあと、管理監督を怠ったとして課徴金に至った流れです（[措置命令](https://www.caa.go.jp/notice/entry/023571/) · [課徴金](https://www.caa.go.jp/notice/entry/029574/)）。

教訓はシンプルです。**抽選がきれいでも、賞品を届けない・遅れると企画全体は信頼を失う**。fair-draw は **抽選機能** なので、発送・履行まではチェックできません。Phase 0 も発送遅延のコンプライアンスを代替しません。幹事側の運用設計が別途必要です。

## 消費者庁に聞いていいのか？

「専門家に聞け」と言われても、**本当に消費者庁に電話していいのか** と迷うのは当然です。

結論：**景品表示法に限れば、事業者向けの相談窓口は公式にあります**。消費者庁は [「事業者がこれから行う企画についての景品表示法に関する御相談」](https://www.caa.go.jp/policies/policy/representation/contact) を **表示対策課 指導係**（代表 03-3507-8800 · 平日 9:30–18:15、12–13 時除く）で受け付けています。自動音声で事業者は **4→1→1** を選びます。

ただし **「気軽にチャット感覚で」ではありません**。同ページでは、相談前に [運用基準・ガイドライン](https://www.caa.go.jp/policies/policy/representation/fair_labeling/) と [景品表示法 Q&A](https://www.caa.go.jp/policies/policy/representation/fair_labeling/faq/) を **まず読むこと**、回答に **相当期間かかる** こと、**実施直前だと間に合わない** ことが明記されています。無制限の無料法律相談でも、個別の合法保証を出す弁護士でもありません。**景表法の所管官庁として、企画の当たりを付ける行政相談** です。

| 誰に | 向いていること | 向いていないこと |
|------|---------------|-----------------|
| **顧問弁護士 · 社内法務** | 告知文案 · 契約 · 業界特例 | 顧問がいない中小は下記を参照 |
| **消費者庁 表示対策課** | **これから行う** 懸賞 CP が景表法上どう見えるか | SNS 運用 · 抽選の公平性 · 発送管理 |
| **消費生活センター · 188** | **消費者** の被害相談 · 苦情 | 幹事の事前企画レビュー（主目的ではない） |

「フォロー RT 懸賞は大丈夫？」のような論点は、**Q&A で自分で当たりを付けたうえで**、企画概要（応募条件 · 景品額 · 販売連動の有無）を1枚にまとめて電話する — という使い方が現実的です。

## 社内法務がない場合 — 現実的な順番

SNS 懸賞の幹事は、マーケ · 広報 · 店舗担当が **法務部なし** で回していることが多い。**「法務に聞け」と言われても聞く相手がいない** — これが現場の前提です。

おすすめは、法務部の代替ではなく **「法務がいないときの最低ライン」** を決めることです。

1. **自分で読む（無料 · 必須）** — [景品表示法 Q&A](https://www.caa.go.jp/policies/policy/representation/fair_labeling/faq/) · [景品規制の概要](https://www.caa.go.jp/policies/policy/representation/fair_labeling/premium_regulation/) · fair-draw Phase 0  
2. **消費者庁に相談（景表法 · 企画前）** — 上記を読んだうえで企画概要を1枚に。**CP 公開の数週間前** を目安に  
3. **スポットの法律相談（リスクが高いとき）** — 景品総額が大きい · 購入条件つき · 業界告示や公正競争規約が絡む · 告知文案を厳密に見てほしい、など。顧問がなくても単発相談は可能。**小規模 CP は 2 で足りる / 大きい CP だけ 3** と線引きする  
4. **社内ナレッジ化** — 2 または 3 の質問文と回答を Wiki に残す  

やってはいけないのは、**法務がいないから ChatGPT だけで go** することと、**発送遅延をツールに任せる** ことです。

## フォロー＋リポスト懸賞は？ — 社内にナレッジを貯める

いまは大小問わず、会社が **X · Instagram · LINE** で懸賞 CP を回しています。幹事が毎回バラバラだと、同じ疑問がゼロから繰り返されます。

- 「**フォロー＋リツイート（リポスト）だけ** のプレゼント企画は問題ない？」  
- 「**X も IG も** フォローしてリポストすると当選確率2倍 — これ公平？」  
- 「景品は無料応募だけど、**割引券の利用に購入が必要** — 取引条件としてどう見る？」  

fair-draw はこれらに **OK / NG と答えません**。Phase 0 では、SNS フォロー・リプライ型の企画フォーマットや、複数 SNS · 口数加重（揉め事カード **P13** 等）を選ぶと **黄フラグとチェックリスト** が出ます。「専門家にこう聞けばよい論点」のたたき台です。

**一度専門家に聞いた内容を社内に残す** のが、ツール単体より長く効きます。

1. fair-draw のチェック結果（フラグ · 入力数値）をスクショ or PDF で残す  
2. 法務 · 顧問（または消費者庁相談）への **質問文と回答** を社内 Wiki / Notion / 共有ドライブに1ページ追加  
3. 次の CP 幹事が「去年のフォローRT懸賞、何を確認したか」を検索できる  

SNS CP の回数が増えるほど、「うちの会社ではこう聞いてこう整理した」という **自社の先例** が資産になります。法務部がなくても、**先例のストック** が次の幹事の法務代わりになります。

## fair-draw でできること · できないこと

| できること | できないこと |
|-----------|-------------|
| 景品上限などの **告示ベース一次チェック**（参考資料） | 法令適合の **最終判定** |
| 揉め事パターンのフラグ · **専門家への問い合わせ推奨** | 晋遊舎型の **履行乖離** の防止 |
| 名簿から当選者を決める · 証跡（txt / PDF / JSON） | 当選品の **発送** · DM · 住所収集 |

「全部解決する懸賞 SaaS」ではなく、**論点整理 · 相談の下準備 · 抽選証跡** にスコープを切っています。

## まとめ

知らないうちに法律を犯すより、企画が範囲内かを専門家に確認するほうが安い — その前提で Phase 0 を置いています。消費者庁には **事前企画の相談窓口がある** が、気軽ではなく Q&A 先読みと **数週間前の相談** が前提です。法務がいない組織ほど、**相談先を決めて、答えを社内 Wiki に貯める** 運用が効きます。

抽選の公平性・証跡3点の話は Zenn 側で書きます。景表法と抽選、両方そろって初めて幹事は安心して説明できます（公開後ここにリンク）。
