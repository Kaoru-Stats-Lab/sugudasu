# 必ず守ること

制約である。改善案で上書きしない。詳細は [`guardrail.md`](guardrail.md)。

## 1. カード UI は変更禁止

- サイズ · 高さ · 横幅 · 余白 — 変更禁止

## 2. 新しい UI コンポーネントを増やさない

- モーダル禁止 · サイドバー禁止 · アコーディオン禁止 · タブ追加禁止

## 3. 検索 UX のみ改善する

対象は次のみ:

- 検索速度（既存エンジンの範囲）
- 検索候補 / チップ
- ゼロ件UX
- 検索チップ

## 4. カテゴリ構造は禁止

- カテゴリ追加禁止 · 小分類禁止 · ツリー禁止 · マルチタグ禁止

## 5. お気に入り仕様は禁止

- 今回は触らない

## 6. 完全ローカル思想を守る

- 検索履歴を保存する場合は **localStorage のみ**
- 通信禁止（クエリの外部送信禁止）

## 7. SEO 文言を書き換えない

- description / blurb 変更禁止（本タスクでは）
- カード名（conceptName）変更禁止

※ 文言タスクは別プロンプト · [`../design/card-guideline.md`](../design/card-guideline.md) と notes 正本に従う。
