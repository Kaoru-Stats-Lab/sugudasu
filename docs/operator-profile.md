## 社名・事業名の公開境界（提督確定 · 2026-06-17）

| 層 | 実社名・事業名 | 例 |
|----|----------------|-----|
| **記者向けのみ** | **可** | PR TIMES 担当者プロフィール（`docs/private/L3-press.md`） |
| **パブリック全般** | **禁止** | サイト · X · PR本文 · changelog · Zenn/note · 取材後の記事引用以外の自社発信 |

- PR TIMES **リリース本文**・開発者コメント・`operator-profile` L0〜L4-public には **社名を出さない**（抽象表現のみ）。
- 二次掲載で記者が社名を書くことは制御不能 — 自社側からは追加で社名を広めない。
- L3-press の文言は提督確認済み（322字）。**変更時は private のみ編集。**
