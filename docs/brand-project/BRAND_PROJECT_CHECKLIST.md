## 更新ルール

ブランドに関する重要な議論が終わるたびに、次を行う。

1. 本チェックリストを更新する
2. `DECISION_LOG.md`へ結論を追記する
3. `OPEN_QUESTIONS.md`を更新する
4. 関連ドキュメントのリンクを確認する
5. 完了したPhase・項目だけをチェックする
6. 新しい未解決事項を追加する

チェックリストへ議論ログ、仕様、判断理由を直接書かない。
