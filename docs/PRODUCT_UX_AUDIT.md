- ✗ 空状態「逆引きを実行」とボタン文言不一致。
- ✗ グラデーションCTA・amber 主色はガイドライン違反。
- ✗ 内蔵辞書12件のみ — 「即答」価値が薄い。
- △ ローカル結果にコピーボタンなし（手数増）。
- ○ 2タブ（その場で使う / AIで広げる）構造は目的に合う。

## STEP 2: 限界値テスト（10+）

1. 空テキスト → alert（要インラインエラー）
2. 空白のみ → 同上
3. 10,000文字 → パフォーマンス・UI崩れ
4. `<script>alert(1)</script>` → 表示は escapeHtml 済み（カード生成時）
5. 絵文字のみ入力 → マッチ0件
6. 1文字トークン `の` → 過剰マッチリスク
7. Geminiタブ未生成でコピー → alert（OK）
8. clipboard API 拒否 → execCommand フォールバック（OK）
9. オフライン → ローカル辞書は動作
10. タブ切替中に再実行 → 状態整合（OK）
11. 全角スペース区切り → トークン分割要改善
12. 辞書0件ヒット → Gemini 誘導メッセージ（OK）
