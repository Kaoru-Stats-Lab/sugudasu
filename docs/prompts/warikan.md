# 傾斜割り勘（warikan.html）

Claude / Gemini 向け要件プロンプト（HTML には埋め込まない）。

---

あなたには、最高峰のフロントエンドエンジニアとして、ログイン不要・完全無料で使える「幹事が得も損もしない、大人の傾斜割り勘メーカー（sugudasu.online/warikan）」のWebアプリ（単一のHTMLファイル）を1発で作成してほしいです。

外部の重いライブラリやサーバーサイドは一切使用せず、HTML/CSS/JavaScriptのみの完全サーバーレスで実装。Tailwind CSS。

## 機能・計算要件

1. **画面構成**: PC左=フォーム、右=結果+LINEテキスト（スマホ縦並び）

2. **入力**
   - 総額（税込）
   - 端数丸め: 100/500/1000/1円
   - 傾斜グループ行（追加・削除可）。初期3階級:
     - 役員・上司 2.0
     - 先輩・主賓 1.5
     - 新人・一般 1.0
   - 係数・人数は編集可

3. **計算**
   - 傾斜倍率×人数で理論上の1人あたり支払額
   - 丸め単位でキリの良い金額に
   - 集まる総額とお会計の差額を明示（幹事の透明性）

4. **プレビュー**
   - 各階級の1人あたり金額を大きく表示
   - LINE/Slack用清算テキスト＋ワンクリックコピー（過不足文言込み）
