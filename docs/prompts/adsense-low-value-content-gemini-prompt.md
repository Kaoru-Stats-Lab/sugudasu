# Gemini依頼用: AdSense「有用性の低いコンテンツ」突破 — SUGUDASU特化コンテンツ拡張

**用途:** Gemini に記事案・本文ドラフトを出させ、**Cursor が実装・事実確認**する  
**更新:** 2026-07-15  
**背景:** AdSense サイト `sugudasu.com` が **要確認 · ステータス詳細「有用性の低いコンテンツ」**。`ads.txt` は承認済み。  
**正本戦略:** [`docs/notes/GUIDES_CONTENT_STRATEGY.md`](../notes/GUIDES_CONTENT_STRATEGY.md)  
**現状監査:** [`docs/notes/ADSENSE_LOW_VALUE_CONTENT_AUDIT_20260715.md`](../notes/ADSENSE_LOW_VALUE_CONTENT_AUDIT_20260715.md)

## 重要（Gemini への先読み注意）

SUGUDASU は **WordPress ブログではない**。無料の**実務ブラウザツール集**＋ `/guides` の実務記事である。  
「日記を15本」「カテゴリを2〜3個に」などの **一般ブログ合格テンプレをそのまま適用しない**。

礼賛・精神論は不要。指定フォーマットのみ。推測は「推測」、不明は「要確認」。

## Cursor 側の受け渡し（実装規律）

1. Gemini 出力の **PACK** を受け取る。  
2. **事実確認**（税率・法令・ツール機能の有無）。虚があれば直してから HTML 化。  
3. `data/guides.json` + `tools/guides/{slug}.html` + 関連ツール FAQ リンク + `changelog` public 1行。  
4. `npm run build:pages` · 必要なら core デプロイ。  
5. 再申請は提督（AdSense 画面）。Agent は再申請ボタンを押せない。
