**提督仮説（突合用）:** ERP を入れない小規模組織で、**Excel / スプレッドシートでガント風工程表**を回している職種・業種（IT 受託 · 建設・設備 · デザイン・制作 等）に、**フルガント SaaS ではなく「連動再計算 + 共有 URL + Excel 出入り」**で入れる余地がある。

```text
あなたは日本の中小企業・現場 PM・フリーランス向けの業務ツール市場に詳しい調査アシスタントです。
礼賛・前置き・記事本文・LP コピーは不要。指定フォーマットの表と箇条書きのみ出力してください。
数値・シェア・料金はソース明記。不明は「要確認」。日本市場を主、海外は参考1行まで。

【我々が検討しているプロダクト（比較の軸 · 誤解禁止）】

- **NOT:** Microsoft Project / Asana / Monday の代替 · クリティカルパス · リソース平準化 · 大規模 ERP · 「ガントチャート SaaS」としての正面競争
- **NOT:** イベント司会の Run of Show（別製品で既に進行中）
- **IS:** 小規模チームが **Excel / Google スプレッドシートで工程表・ガント風表**を運用している領域への **軽量クラウド**
  - 1行の日付・所要・タスク変更で **後続が連動再計算**（縦型タイムライン / 工程表）
  - **CSV/Excel からの取込** · **TSV/Excel への書き戻し**（Copy-first）
  - **共有 URL** で版を提示（PDF/ファイル再送の代替）· 本番中の大きな Checkout は避ける想定
  - 価格帯: イベント単位の少額チケット（月額 PM ツールより安いイメージ）
  - ブランド: SUGUDASU Sync（幹事向け無料コアとは別ライン）

【依頼1】Excel でガント・工程表を作っている職種 × 業種マトリクス

**最低 15 行**（日本で現実的に多い順）。列:

| 列名 | 内容 |
|------|------|
| 業種（大分類） | 例: 建設 · 受託 IT · 広告制作 |
| 職種・役割 | 例: 現場監督 · PM · ディレクター |
| 組織規模の目安 | 例: 5–30人 · 個人事業主 |
| Excel 工程表の典型用途 | 週次 / 日次 / 案件単位 / クライアント提示 |
| ERP・本格 PM を入れない理由（仮説） | コスト · 習得 · 案件の小ささ · 慣習 |
| 更新頻度 | 日次 / 週次 / マイルストーン時 |
| 共有相手 | 社内 / 下請 / クライアント / 現場 |
| Excel の具体的ペイン | 式崩れ · 版管理 · スマホ · 再送 等 |
| 既に使っている代替（あれば） | Notion · スプレッドシート共有 · 紙 等 |

【依頼2】垂直別 — 勝ち筋スコアリング

依頼1から **上位 8 垂直**を選び、次の列で **1–5 点**（5=最良）を付け、合計でランク付け:

| 評価軸 | 意味 |
|--------|------|
| Excel 依存度 | 表計算が正本であるほど高い |
| ERP 空白 | 大手 PM/ERP が入りにくいほど高い |
| 連動再計算の価値 | 遅延・変更で後続がずれるほど高い |
| 共有・版混乱の痛み | PDF/ファイル再送が多いほど高い |
| 少額課金の通りやすさ | 稟議なし個人/小チームで決まりやすいほど高い |
| 競合の強さ（逆点） | Asana 等が既に標準 → **低い点** |
| 日本語ニッチ SEO 余地 | 「工程表 エクセル」系で LP を切りやすいほど高い |
| 我々エンジン適合 | 縦1本の時系列・所要連鎖で足りるほど高い |

列: 垂直名 | 各軸スコア | 合計 | 1行理由 | **PoC 推奨順位（1–8）**

【依頼3】検索・言語 — 垂直ごとのキーワード（ガント正面は避ける）

各垂直 **3 Primary + 5 Secondary**（日本語）。**主語に「ガントチャート」だけを置かない** 方針。

列: 垂直 | Primary（商用/課金向け）| Secondary（無料流入）| **避けるキーワード**（コモディティ・誤認）

例の方向性（そのまま使わなくてよい）:
- 工程表 エクセル テンプレート
- 制作 スケジュール 共有
- 工事 日報 工程（※建設は専門 ERP と混同注意）

【依頼4】競合・代替マップ（垂直ごと · 8 行以内）

列: 垂直 | 現場のデフォルト（Excel 以外）| 乗り換え障壁 | **我々が刺さる隙間（1行）** | 刺さらない理由（正直）

**禁止:** 「日本唯一」「完全代替」· 未確認の市場規模

【依頼5】Excel 出入り — 現場が期待する形式

列: 垂直 | 典型的な列（開始日/終了日/工数/担当/備考 等）| 取込（貼付/CSV/xlsx）の現実 | 出力（TSV/xlsx/PDF）の優先度 | **MVP で捨てていい列・機能**

【依頼6】勝ち筋の結論 — 3 段落（箇条書きのみ）

【依頼8】リスク・誠実開示（我々向け）
```
