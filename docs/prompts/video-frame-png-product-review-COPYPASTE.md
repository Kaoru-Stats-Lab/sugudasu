# ローカル動画コマ抜き（PNG）— プロダクト判断用プロンプト（COPYPASTE）

**目的:** 「ニッチすぎないか」を他AIに辛口で問う。実装仕様の細部より **採否 · ペルソナ · 代替手段** を固める。  
**ロードマップ候補:** `data/roadmap.json` · `mask-video-frame-capture`  
**関連 OUT:** Tango 型自動手順書 · GIF 化（同 roadmap 対象外レーン）

**使い方:** 下の「プロンプト本文」を Gemini / ChatGPT / Claude / Grok 等にそのまま貼る。追加資料は不要（本文に背景を含む）。
