# SUGUDASU 赤入れ — FAQ 正本

**公開:** `tools/annotate.html` の FAQ / JSON-LD と同期すること。

---

## PDFは編集できますか？

いいえ。PDF **編集** ツールではありません。ページを **画像として開き**、黒塗り · 矢印 · 枠を付けます。未編集ページは **元PDFのまま** 保持して出力します。

## PDF記入（Paper First）との違いは？

[PDF記入](../pdf-fill/overview.md) は書類を **完成** させる（入力 · 署名 · 保存）。  
赤入れは **提出 · 共有前** に、必要な場所だけ整えて **共有できる状態** にする。Copy First。

## 画像はサーバーに送信されますか？

いいえ。ブラウザ内の Canvas だけで処理します。

## ぼかしと黒塗りはどちら？

マニュアル向けは **ぼかし**。パスワード · ID · 金額は **黒塗り** または **モザイク** を推奨します。

## 画像切り出し · 透かしとの違いは？

| ツール | 役割 |
|--------|------|
| 赤入れ | 共有できる状態へ整える（隠す · 囲む · 指す） |
| 画像切り出し | サイズ合わせ |
| 透かし | 所有 · 社外秘表示 |

併用例: 赤入れ → 切り出し → 透かし
