## テキスト注釈 — Reject（2026-07-24）

| 項目 | 決定 |
|------|------|
| 自由記述テキスト | **不採用** — UI から削除 |
| 理由 | 矢印 + テキスト + 吹き出し = **特定カテゴリ（Skitch 型）の視覚言語一式**。JTBD「共有できる状態へ整える」から外れる |
| 代替 | 矢印 · 枠 · 楕円で視線を導く。固定文言は **スタンプ**（サンプル / ダミー / テスト）のみ |

## Visual Grammar Rule（2026-07-24）

| 項目 | 決定 |
|------|------|
| 原則 | 普遍的視覚記号は可 · 特定プロダクトを想起させる視覚言語一式は不可 |
| 採用可 | 矢印 · 四角 · 円 · ハイライト |
| 不採用 | 吹き出し · テキスト注釈 · 手書きペン · スタンプ大量 · リッチコメント |
| 正本 | `philosophy.md` — Clip Stash Non-Goals と同型の判例 · **CASE-2026-003** · [`../../legal/CASE_LAW.md`](../../legal/CASE_LAW.md) |

## 矢印形状 — HOLD（2026-07-24）

| 項目 | 決定 |
|------|------|
| 形状変更 | **まだやらない** |
| 理由 | 矢印は PowerPoint · Excel · スライド等と共有された視線誘導記号。**矢印単体 ≠ 特定プロダクト** |
| 優先 | **期待値コントロール**（コピー · ポジショニング）。見た目は後からでも変えられる |

## ポジショニング — GO（2026-07-24）

| 項目 | 決定 |
|------|------|
