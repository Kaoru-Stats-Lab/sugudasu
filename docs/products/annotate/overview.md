# SUGUDASU 赤入れ — 概要

**id:** `annotate`  
**プロダクト名:** SUGUDASU 赤入れ  
**ステータス:** v1 · 2026-07-24  
**実装:** `tools/annotate.html` · `assets/annotate-app.js` · `assets/annotate-engine.js`

---

## 一文

**提出 · 共有前に、必要な場所だけ整える** — 隠す · 囲む · 指す · コピー。

---

## 目的

| やる | やらない |
|------|----------|
| 黒塗り · ぼかし · モザイク | OCR / 顔の自動検出 |
| 矢印 · 枠 · 楕円 | PDF の提出完成（→ [PDF記入](../pdf-fill/overview.md)） |
| 画像 · PDF 入力 · Copy First 出力 | Acrobat 代替 · レイヤー編集 · クラウド保存 |
| PDF は **編集ページのみ** 画像化して出力 | 全ページの無差別書き換え |

---

## PDF記入との境界

| | PDF記入 | 赤入れ |
|---|---|---------|
| JTBD | 書類を **完成** させる | **共有できる状態へ整える** |
| 典型 | 署名 · 記入 · 黒塗りして提出 | ここ直して · この数値 · 確認お願い |
| 出力 | 提出用 PDF（焼き付け） | PNG / JPEG / 部分 PDF · **コピー優先** |

---

## 入力 · 出力

- **入力:** PNG / JPEG / WebP / GIF / PDF（最大 25MB · ページ上限あり）
- **出力:** クリップボード（PNG）· PNG · JPEG · PDF（編集ページのみラスタ · 未編集は copyPages）
- **処理:** ブラウザ内 Canvas · pdf.js / pdf-lib

---

## 誠実線

- 画像は **当社サーバーへ POST しない**
- **塗り残しは利用者責任** · 出力前に目視確認
- パスワード · ID · 金額は **黒塗りまたはモザイク** を推奨（ぼかしは限界あり）
- 「完全匿名化」「100%安全」は **禁止**
